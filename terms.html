欢迎使用「花迹 Floriary」。本协议是你与开发者 Eric Liu（以下简称「我」或「开发者」）之间关于使用花迹应用的法律协议。安装、注册或使用本应用，即表示你已阅读、理解并同意本协议的全部条款。

如不同意本协议，请立即停止使用并从你的设备上卸载本应用。

一、服务说明

花迹是一款植物成长记录应用，提供以下功能：

  • 花园与植株档案管理
  • 成长记录（文字备注、照片、养护标签）
  • 植株标签（自定义、带颜色、可按标签筛选）
  • 绿植资产统计（入手日期 / 价格 / 价值）
  • 养护提醒与养护中心
  • 数据统计、热力图与成长洞察
  • 成长档案 / 纪念册导出
  • AI 识图（拍照认植物）与物种纠错
  • 结构化数据跨设备云同步（所有账号用户均包含）
  • 反馈与评价
  • 会员增值功能（详见第四节）

二、账号注册与使用

2.1 注册要求

  • 你必须提供真实、有效的信息（如邮箱地址），并确保信息准确。
  • 每个邮箱地址或第三方账号（Apple ID 等）只能注册一个花迹账号。
  • 你须年满 14 周岁（或在监护人指导下，根据所在地法律规定）方可注册使用。

2.2 账号安全

  • 你负责保管账号凭证（邮箱密码、登录设备），不得将账号转让、出借或共享给他人。
  • 如发现账号被盗用，请立即通过「忘记密码」功能重置凭证，并联系开发者邮箱。
  • 因账号凭证泄露导致的损失，由账号持有人自行承担，但若因我方服务本身的安全漏洞导致的除外。

2.3 多种登录方式

你可以在「我的 → 账号与安全」中为同一账号绑定多种登录方式（邮箱、Apple ID 等）。每种绑定均需经过验证，同一邮箱/Apple ID 不可同时绑定多个花迹账号。账号必须保留至少一种有效的登录方式，不可全部解绑。

三、你的内容

3.1 内容归属

你在花迹中创建的花园、植株、成长记录、照片、植株标签、资产信息等内容（统称「用户内容」）的知识产权归属于你。

3.2 授权

为向你提供服务（包括数据存储与跨设备同步），你授权我在技术上必要的范围内处理、存储和传输你的内容。其中，付费会员的照片同步通过你自己的 iCloud 由 Apple 完成，花迹不持有该等照片。此授权不涉及任何商业用途，不向第三方转授。

3.3 内容责任

你对自己上传或创建的内容承担责任，并保证：

  • 内容不侵犯他人知识产权、隐私权或其他合法权益；
  • 内容不违反中华人民共和国及你所在地的法律法规；
  • 不利用本应用存储、传播违法、有害或侵权内容。

3.4 反馈、物种纠错与识图贡献

  • 当你提交反馈或物种纠错建议（如品种名称、养护信息、配图修正等）时，你授予花迹免费、非独占地使用该等内容以改进产品与完善植物库的权利。
  • 当你同意识图训练数据采集后，你的识图照片与最终确认结果可用于改进花迹自有的植物识别模型（详见《隐私政策》）。
  • 上述使用均经去标识化处理，不公开你的账号身份；你注销账号时，相关贡献数据将按《隐私政策》清理。

四、付费服务

花迹提供「会员」付费增值服务，解锁一系列高级权益。

4.1 会员权益

会员解锁以下功能权限：

  • 无限花园数量（免费版上限 3 个）
  • 每个花园无限植株数量（免费版每园上限 20 株活跃植株）
  • 绿植图片 iCloud 同步——照片存入你自己的 iCloud 容器，跨设备同步、换机自动恢复（免费用户图片仅保存在本机本地）
  • 完整统计历史与数据洞察（免费版仅展示近 1 个月热力图）
  • 成长档案 / 纪念册导出（无水印、全量、高清长图 / PDF）
  • 识图永久无限次（免费用户为限时免费，推广期结束后回落为有限次数）
  • 养护任务批量完成（一键批量完成养护待办）
  • 无限植株标签（自定义带色标签、可按标签筛选）

4.2 自动续费说明

  • 自动续费订阅将在当前订阅周期结束前 24 小时内自动通过你的 Apple ID 账户扣费续订，续费后订阅期顺延。
  • 你可以随时前往「iPhone 设置 → [你的名字] → 订阅」管理或关闭自动续费。
  • 关闭自动续费后，订阅权益在当前已付费周期结束前仍然有效，到期后不再续订。
  • 含免费试用期的订阅：试用期结束后，如未在试用期内取消，将自动扣费续订。

4.3 退款

所有购买均通过 Apple App Store 完成结算，退款事项请直接联系 Apple（reportaproblem.apple.com）。开发者不直接处理退款。

4.4 价格变更

开发者保留调整订阅价格的权利。价格调整将提前在应用内告知，且不影响当前已付费订阅周期内的费用；下一个计费周期起按新价格执行。

4.5 会员到期与超限策略

会员到期（未续费）：

  • 超出免费版额度的花园/植株/记录数据保留，但进入只读状态（无法新增或编辑超限部分，已有数据可查看）。
  • 完整统计历史、数据洞察、纪念册导出、批量养护、识图无限、无限标签等会员功能回退到免费版能力范围。
  • 已存入 iCloud 的历史照片保留可读；到期后新照片回落本机本地，不会突然删除你任何照片。重新订阅会员后，高级功能权限与 iCloud 同步即时恢复。
重要：到期不会立即删除你的数据；数据保留，仅限制新增/编辑操作。

六、账号注销

你可以随时通过「我的 → 账号与安全 → 注销账号」申请注销。

  1. 订阅提醒：注销账号不会自动取消你的 Apple 会员订阅扣费。应用会在注销前明确提示，并提供跳转「App Store 订阅管理」的入口，请按需自行前往取消。
  2. 二次验证：需输入当前密码（或其他已绑定的验证方式）确认是本人操作。
  3. 冷静期：提交注销申请后，账号进入 7 天冷静期。冷静期内你仍可登录并撤销申请。
  4. 数据处理：冷静期结束后，你的账号数据将被软删除和匿名化；识图照片、识别记录、反馈数据及服务器上的关联图片进入清理流程；本机本地照片由设备本地清除；存入 iCloud 的照片由你的 iCloud 管理（注销不会代你删除 iCloud 中的照片）。
  5. 不可恢复：冷静期结束、数据删除后，数据无法恢复。

七、禁止行为

你在使用本应用时不得：

  • 破解、反编译、逆向工程本应用；
  • 绕过付费限制（如冒用他人账号或伪造收据等）；
  • 利用本应用存储、传播违法内容（含色情、暴力、涉政敏感内容等）；
  • 对服务器发起 DoS/DDoS 攻击或其他干扰服务的行为；
  • 爬取、批量下载其他用户的数据；
  • 以任何方式滥用识图 AI 服务（含上传与植物识别无关的有害内容）；
  • 提交虚假或恶意的反馈 / 物种纠错信息以干扰运营。

违反上述规定的，我有权终止你的账号服务，情节严重的将依法追究法律责任。

十、免责声明

  • 服务可用性：本应用按「现状」提供服务，不承诺服务永不中断或无故障。因不可抗力、第三方服务商故障（如 Apple、云服务商、识别服务商）、网络问题等导致的服务中断，不承担赔偿责任，但会尽合理努力恢复服务。
